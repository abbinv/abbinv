### Prerequisites
- Node.js installed
- Firebase CLI installed globally: `npm install -g firebase-tools`

### Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Login to Firebase:
   ```bash
   firebase login
   ```

3. Initialize Firebase project:
   ```bash
   firebase init hosting
   ```

### Development
```bash
npm start
```
Serves the site locally at http://localhost:5000

### Deployment
```bash
npm run deploy
```
